vue-calendly

Vue.js component for using Calendly widget.

View on GitHub Download .zip Download .tar.gz

Installation

npm install vue-calendly --save-dev

1) Download from NPM

Usage

1) Install the Vue Calendly Component: 

import Vue from 'vue';
import VueCalendly from 'vue-calendly';

Vue.use(VueCalendly);

2) Use the Vue Component in your own component: 

<vue-calendly url="your Calendly URL" :height="600"></vue-calendly>
